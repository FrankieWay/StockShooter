## 二、新闻与行情数据来源（可爬/可查）

- **快讯/要闻**：财联社、东方财富快讯、同花顺要闻、新浪财经、金十数据。
- **宏观与政策**：新华网财经、人民网财经、证券时报、第一财经、央行/发改委官网。
- **个股与筹码**：东方财富（股东户数、户均持股）、同花顺、交易所公告、巨潮资讯（年报/季报）。
- **行业分类**：申万行业分类、东方财富行业板块、同花顺概念板块。

## 三、筹码与流动性指标说明

- **股东户数下降**：通常表示筹码从散户向大户集中，利于拉升。
- **户均持股数/流通市值上升**：与户数下降同向，可作辅助。
- **前十大流通股东持股比例**：比例高且稳定表示筹码集中。
- **日成交额**：建议筛选日成交额 ≥ 5000 万（可依市场环境微调），保证可交易性。

以上指标若无法获取，在清单中注明“未验证筹码/流动性”，避免臆测。

## 四、Tushare API（可选）

当用户提供 Tushare token 时，可用于步骤 3 的个股筛选与步骤 4 的流动性/筹码校验，数据更统一、易批量处理。

### Token 与环境

- 用户需在 [tushare.pro](https://tushare.pro) 注册并获取 token。
- **本仓库**：若项目根目录存在 `.env` 且已设置 `TUSHARE_TOKEN`，则优先使用（执行脚本前可加载 `.env`，如 Python 的 `python-dotenv`）；否则读取系统环境变量 `TUSHARE_TOKEN`。模板见 `.env.example`，复制为 `.env` 并填入 token 即可。
- **不要在 skill 或代码中写死 token**；仅从环境或 `.env` 读取。

### 推荐接口与用途

| 用途 | 接口 | 说明 |
|------|------|------|
| 股东户数、户均持股 | `stk_holdernum` | 按季度，用于判断筹码是否集中（户数降、户均持股升） |
| 日行情、成交额 | `daily` 或 `daily_basic` | 筛选日成交额 ≥ 5000 万、看放量 |
| 申万行业分类 | `index_member` / 申万行业成分 | 按产业取成分股，再叠加筹码与成交额筛选 |
| 概念/板块成分 | 概念板块成分接口 | 热点映射到概念后取成分股 |

### 使用约定

- 未提供 token 时，不调用 Tushare，仅用网页搜索与公开数据，清单中注明“未使用 Tushare”。
- 使用 Tushare 时，在最终清单的“数据与免责声明”中注明“个股及筹码数据来源：Tushare”。
- 注意 Tushare 的积分与调用频率限制，按需取数，避免单次拉取过多。
